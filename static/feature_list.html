Feature List

Last Updated: 11/10/23

Current Features

  1. User can choose a language from a drop-down list of a few offered languages (Spanish, French, German, Norwegian)
  2. User can view a list of grouped words/phrases in a foreign language along with their English translations
  3. User can use flashcards to help memorize these new words/phrases
  4. User can test their knowledge by taking a randomly-generated quiz of 10 questions
  5. User can search for terms in the dictionary tab to see their English definition and translations into all the offered languages
  6. User can filter by different phrase groups (introductions, numbers, etc.) when learning their words and taking quizzes
  7. User will see a cycling list of greetings from various languages upon reaching the site

Features to be Implemented

  1. User will be able to optionally create an account
  2. User's progress and previous quiz scores will be saved if they are currently logged in
  3. User can play a matching game in which they select a language and category(ies). Words of a foreign language will be on one side, and their translations will be on the other. Users will be able to drag connecting lines to match the correct translations.
  4. At least one more language will be added to the current list of learnable languages

Possible Future Features

  • More learnable languages
  • Audio clips that pronounce the words
  • User could select a primary language to learn and not have to keep re-selecting what language they want to learn across different activities
